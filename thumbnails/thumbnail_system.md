# Truth Explains thumbnail system

Goal
Readable on a phone in 1 second.

## Layout

Text placement
Left aligned or center aligned.
Pick one and keep it consistent.

Visual elements
One supporting image only.
No clutter.
No extra icons.

## Text rules

Word count
3 to 4 words max

Style
Sentence case
No punctuation
No quotes

Examples
What cybersecurity is
You are not behind
Before you rush
Tools are not enough
Start here first

## Image rules

Avoid
Hacker hoodies
Green code rain
Busy collage thumbnails
Tiny objects that cannot be seen on a phone

Prefer
Calm human imagery
Hands on keyboard
Desk and notebook
Abstract tech visuals
Servers or data flow

## Consistency rules

Same font every time
Same placement pattern every time
Same background color family every time

If it feels too loud, it is off brand.
