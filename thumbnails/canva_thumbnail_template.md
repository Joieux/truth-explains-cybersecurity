# Canva thumbnail template for Truth Explains

Goal
A calm, consistent thumbnail that is readable on a phone in one second.

Canvas size
YouTube thumbnail
1280 x 720

Safe area rule
Keep all text and key visuals inside the center 80 percent of the canvas.
Avoid the extreme edges.

## Layout

Text block
Place the text block on the left side.
Text should occupy roughly the left 40 percent of the canvas.

Image
One supporting image on the right side.
Image should occupy roughly the right 60 percent of the canvas.
No collage.
No multiple icons.

Background
Solid warm charcoal or deep brown.
Optional subtle texture at very low intensity.

## Text styling

Font
Inter or Montserrat

Weight
Semi bold

Case
Sentence case

Word count
3 to 4 words max

Line breaks
Prefer two lines.
Example
What cybersecurity
really is

Do not use punctuation.
Do not use quotation marks.

## Text size guidance

Primary text size
Large enough to read on a phone.
If you zoom out to thumbnail size and cannot read it instantly, increase size.

Stroke and shadow
Avoid heavy outlines.
If contrast is low, use a very subtle shadow or a slightly darker overlay behind the text.

## Color usage

Text color
Soft cream or off white

Accent color
Muted gold or dusty terracotta
Use for a small underline or a tiny corner bar only.
Never use accent on the main text.

## Examples of strong thumbnail text

What cybersecurity is
You are not behind
Before you rush
Tools are not enough
Start here first

## Consistency check

Same font every time
Same placement every time
Same calm background family
If it feels loud, it is off brand
