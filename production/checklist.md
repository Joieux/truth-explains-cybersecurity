# Episode Checklist

Copy this for each episode. Check off as you go.

## Pre-production
- [ ] Script written and reviewed
- [ ] Script reads naturally out loud — no robotic phrases
- [ ] YouTube title, description, and tags drafted

## Narration
- [ ] ElevenLabs narration generated
- [ ] Audio downloaded and saved to audio/epXX_narration.mp3
- [ ] Total duration noted (in seconds)

## Clip generation
- [ ] Character anchor pasted into every Runway prompt
- [ ] All story scenes generated (2–3 takes each, best selected)
- [ ] Clips saved to clips/epXX/ with correct file names
- [ ] Library clips confirmed in clips/library/ (do not regenerate)

## Edit
- [ ] DaVinci project created
- [ ] All assets imported
- [ ] Narration placed on A1
- [ ] Clips cut to narration with J-cuts
- [ ] Intro and outro cards in place
- [ ] Full playback — no audio gaps, no placeholder clips showing

## Color and audio
- [ ] Warm grade on Auntie Truth scenes
- [ ] Scammer scene cooler than Auntie Truth scenes
- [ ] Narration sitting at -12 dB
- [ ] Limiter on master bus

## Export and upload
- [ ] Exported to renders/epXX_truth_explains_final.mp4
- [ ] Uploaded to YouTube
- [ ] Title, description, tags, thumbnail set
- [ ] Added to playlist
- [ ] End screen added
- [ ] Published or scheduled
