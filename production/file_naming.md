# File Naming

One rule: lowercase, underscores, two-digit episode numbers. No exceptions.

## Scripts
scripts/epXX_short_title.md
Example: scripts/ep01_what_cybersecurity_is.md

## Narration audio
audio/epXX_narration.mp3
Example: audio/ep01_narration.mp3

## Story scene clips
clips/epXX/epXX-sX-slug.mp4
Example: clips/ep01/ep01-s1-door.mp4
         clips/ep01/ep01-s2-surface.mp4

## Library clips (Auntie Truth — generate once, reuse always)
clips/library/at-lib-XX.mp4
Example: clips/library/at-lib-01.mp4

## Final renders
renders/epXX_truth_explains_final.mp4
Example: renders/ep01_truth_explains_final.mp4

## Prompts
prompts/epXX/epXX_scene_prompts.md
prompts/library/library_clip_prompts.md
