# File naming standard

Goal
Keep the project organized and searchable forever.

General rules
Use lowercase
Use underscores
Avoid special characters
Avoid long filenames
Use two digit episode numbers

## Scripts

Format
scripts/video_XX_title.md

Examples
scripts/video_01_what_cybersecurity_is.md
scripts/video_02_not_a_genius.md
scripts/video_03_skills_before_tools.md

## Audio exports

Format
audio/video_XX_title_ai.wav

Examples
audio/video_01_what_cybersecurity_is_ai.wav

## Project files

Format
video_projects/video_XX_title_project.ext

Examples
video_projects/video_01_what_cybersecurity_is_project.drpr
video_projects/video_01_what_cybersecurity_is_project.capcut

## Final renders

Format
renders/video_XX_title_final_1080p.mp4

Examples
renders/video_01_what_cybersecurity_is_final_1080p.mp4

## Shorts

Format
shorts/short_XX_topic_version.mp4

Examples
shorts/short_01_password_reuse_v1.mp4
shorts/short_01_password_reuse_v2.mp4

## Thumbnails

Format
thumbnails/video_XX_title_thumb.png

Examples
thumbnails/video_01_what_cybersecurity_is_thumb.png

## Dates

If you need dates, add them at the end in this format
YYYY_MM_DD

Example
renders/video_01_what_cybersecurity_is_final_1080p_2025_12_29.mp4
