# Aurora Reid — Color Wheel Jacket Prompts (ChatGPT)
One character, one outfit. Only the windbreaker color and LED accents change.
Workflow: new ChatGPT conversation → upload `branding/character_reference/auntie_truth_ref_v2.jpg`
→ send the SESSION SETUP prompt → then one color prompt at a time.
Save results to `branding/character_reference/` as `aurora_red.png`, `aurora_blue.png`, etc.

---

## SESSION SETUP — send first, with the reference image attached

```
This image is my locked character reference: Auntie Truth (Aurora Reid).
Study it carefully. In every image I ask for in this conversation, keep her
EXACTLY consistent with this reference: same face, same warm brown skin,
same reddish-bronze locs with gold loc cuffs, same bold plain black
square-frame glasses (no decorations on the frames), gold hoop earrings,
layered gold chains with a USB drive pendant, stacked gold rings and
bracelets, purple nails, fitted black graphic tee, wide-leg black cargo
pants, chunky white sneakers, and her studio: record crates beside server
racks, vintage synthesizer, multiple monitors, neon sign, plants everywhere.

Do NOT add: kanji or Japanese-character patches, ankh pendants, or any
decoration on her glasses.

The ONLY thing that will change between images is the COLOR of her oversized
vintage windbreaker (worn open, subtle circuit-board pattern panels) and the
studio's LED accent lighting, which always matches the jacket.

Art style, locked: anime, MAPPA-style linework, flat cell-shading, subtle
neon accents, cinematic lighting. Confirm you've locked the character, then
wait for my first prompt.
```

## Tips

- One prompt per message. If she drifts (glasses change, jacket closes,
  patches appear), reply: "Stay locked to the reference — only the jacket
  color changes" and regenerate.
- Generate 2–3 takes per color, keep the best.
- Check before saving: plain black square frames, no kanji patch, no ankh,
  USB pendant present, white sneakers.
