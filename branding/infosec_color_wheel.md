## The wheel, translated to Truth Explains

The industry uses the color wheel for security *teams*. We use it for *lessons*.
Each color answers a different viewer question.

| Color | Industry meaning | Our meaning | Viewer question it answers |
|-------|-----------------|-------------|---------------------------|
| 🔴 Red | Offense / attackers | **Know the attacker** | "How do they get people?" |
| 🔵 Blue | Defense | **Protect yourself** | "What do I do about it?" |
| 🟡 Yellow | Builders | **Set it up right** | "How do I build my setup?" |
| 🟣 Purple | Red+Blue collab | **Both sides / channel default** | "How does this all fit together?" |
| 🟢 Green | Blue+Yellow (DevSecOps) | **Automate your safety** | "How do I make this run itself?" |
| 🟠 Orange | Governance, risk, training | **Your rules and rights** | "What are my rights here?" |

These tags are editorial lenses on the standard industry model. One episode,
one color. If an episode genuinely spans the wheel, it's purple.

*(Some versions of the wheel add a white team for compliance/audit. We hold a
white jacket in reserve but it's not part of the core system.)*

## Social asset color coding

Rules (consistent with `design_system.md` — these REPLACE the gold/terracotta accent
when content is color-tagged):

1. **Background never changes.** Warm charcoal (#2C2420), every asset, every color.
2. **Text never changes.** Soft cream / off-white.
3. **The team color appears in exactly three places:**
   - Auntie Truth's windbreaker (in any character art)
   - One accent bar or border element
   - The small color-dot badge (bottom corner, with team name, e.g. "● RED — Know the Attacker")
4. One color per asset. No gradients, no second accent.

**Carousel template:** cover slide = character art (jacket in team color) + title;
content slides = charcoal background, cream text, single accent bar in team color;
final slide = recurring CTA in the same color.

**Thumbnail template:** character left or right third (jacket = episode color),
title text in cream, single accent element. Episode color = jacket color, always matched.

**Suggested accent values** (tune to taste once first assets are made):
red #C0392B · blue #2E6FBF · yellow #D9A441 · purple #7D5BA6 ·
green #3E8E5A · orange #C96A2B · (white reserve #EDE6DC)

## Why this works

Viewers learn the code without being taught it: red thumbnail = scam warning,
blue = protection how-to, yellow = setup day. The character never changes, so
recognition compounds — but the wardrobe tells them what's for dinner.
